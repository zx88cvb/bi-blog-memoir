# 1. 前置条件

1. 拥有 Ultra mobile PayGo 号码
2. Visa 或 万事达信用卡
3. Apple美区账号(iPhone用户), 安卓用户可在Google Play上下载

## 3.1 绑定支付方式

  <img

点击主页中图中红色区域 PayGo Wallet +号 跳转到钱包页面。

选择你要充值到金额 点击 + 号

    src="https://r2.haydenbi.com/post/ultra-mobile-payment-credit-card.png"

后面选择支付方式，这里我们选择信用卡支付

输入你的信用卡卡号、到期时间、安全码

最后一栏Zip Code为美国的邮政编码，这里我填写的是90016 位于Los Angeles, California

Zip Code填写的不同最后充值金额的税也会有所不同。

## 3.2 充值

绑定支付方式后就可以回到PayGo Wallet页面选择对应金额充值即可，充值成功后钱包里面会增加对应的金额后续扣费时会优先扣钱包里的余额。

可以看到通过官方充值比第三方充值服务便宜不少。
